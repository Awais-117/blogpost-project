# 🚀 Django Blog Platform 

A modern, full-stack **blogging platform** built with Django where users can write, explore, and interact with content — inspired by platforms like Medium.

## ✨ Core Features

### 📝 Content Creation

* Create, edit, and delete blog posts
* Draft vs Publish system
* Add featured images
* Clean writing-focused UI

---

### 🔥 Smart Feed System

* **For You** → Shows all posts
* **Trending** → Based on views & engagement
* **Featured** → Posts from followed users

---

### ❤️ Engagement System

* Clap (like) system with real-time update
* View count tracking
* Smooth UI animations

---

### 👤 User System

* User registration & login
* Profile system
* Follow users feature
* Personalized content feed

---

### 🎨 UI / UX (Professional Level)

* Medium-inspired design
* Responsive across all devices 📱
* Smooth transitions & hover effects
* Clean card-based layout
* Optimized typography for reading

---

### 📄 Pages Included

* Home Page (dynamic feeds)
* Detail Page (blog reading experience)
* Create Post Page
* Profile Page
* Authentication Pages

---

## 🛠 Tech Stack

* **Backend:** Django
* **Frontend:** HTML, CSS, Bootstrap
* **JavaScript:** Fetch API (AJAX interactions)
* **Database:** SQLite

---

## ⚙️ Run Locally

```bash
git clone https://github.com/Awais-117/blogpost-project.git
cd blogproject
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
